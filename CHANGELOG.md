## 2026-04-24 - 总聊天框 + 自动学习规则系统

### 🆕 新增功能

#### 1. 总聊天框（宏观调控）
- **位置**：左侧面板最上方，生成模式选择之上
- **功能**：
  - 输入通用指令，如"注意时间范围要完整，11:30-13:00 不是只测到 12:00"
  - Ctrl+Enter 快捷发送
  - 聊天历史保存到 localStorage
  - 每次生成时自动追加到需求描述传给 AI

#### 2. Harness 自动学习系统

**三重质量保障：**
1. **AI 自审**：规则入库前自动审核，过滤明显错误的规则
2. **自动淘汰**：使用 5 次后有效率 < 30% 自动禁用
3. **手动管理**：规则管理页面可启用/禁用/删除规则

**自动学习触发：**
- 用户给反馈（👍/👎）后自动触发
- 显示学习进度和结果
- 入库 / 过滤数量透明可见

**新增 API：**
| API | 方法 | 功能 |
|-----|------|------|
| `/api/rules/learn` | POST | 自动学习规则 |
| `/api/rules/detail` | GET/PATCH/DELETE | 规则管理 |
| `/api/rules/cleanup` | POST | 清理已淘汰规则 |
| `/api/rules/stats` | GET | 获取统计信息 |

#### 3. 首次使用引导
- 页面加载时检测首次访问
- 展示 4 大核心功能
- 3 步快速上手教程
- 记录到 localStorage，之后不再显示

#### 4. 用例管理功能

**删除用例：**
- 每条用例右侧有 🗑️ 删除按钮
- 可单独删除或多选后批量删除

**继续生成：**
- 点击「➕ 继续生成」按钮
- 在现有用例基础上追加生成更多
- 可补充说明（如"补充边界值用例"）
- 需求和示例保持不变

#### 5. 单条用例编辑与反馈

**单条编辑（✏️）：**
- 点击用例右侧「✏️」按钮
- 弹出编辑框，可手动修改任意字段
- 保存后记录修改历史

**单条反馈（💬）：**
- 点击用例右侧「💬」按钮
- 输入问题描述/修改建议
- AI 自动修改这条用例
- 保存修改历史

**修改过程记录：**
```javascript
{
  case_id: "TC_003",
  type: "edit" | "feedback",
  timestamp: "2026-04-24T12:00:00",
  original: { ... },  // 原始内容
  modified: { ... },  // 修改后内容
  user_feedback: "预期结果不对"  // 用户反馈（仅 feedback 模式）
}
```

**修改后通过学习：**
- 用例修改后点「✅」
- 系统自动学习"如何从错误改进到正确"
- 学习内容包含：原始用例 + 修改后的用例 + 用户反馈
