# 项目依赖说明

## Python 版本
- 建议：Python 3.8+
- 最低：Python 3.7+

---

## 第三方依赖（**零依赖！**）

无需安装任何第三方包！

---

## Python 标准库（**全部使用内置库**）

所有功能使用 Python 内置标准库：

| 模块 | 用途 |
|------|------|
| http.server | Web 服务 |
| urllib | HTTP 请求 |
| json | JSON 解析和保存 |
| csv | CSV 导出 |
| difflib | 字符串相似度计算 |
| uuid | 会话 ID 生成 |
| pathlib | 文件路径操作 |
| socketserver | 多线程 Web 服务 |

---

## 浏览器依赖（可选，用于字体）

- Google Fonts（可选，网络加载，失败自动回退到系统字体）
  - Noto Sans SC
  - JetBrains Mono

---

## 安装说明

```bash
# 无需安装第三方包！

# 直接运行服务
python web/app.py
```

---

## 注意事项

- 无需安装 Flask/Django/等 Web 框架（使用 http.server）
- 无需安装 requests（使用 urllib）
- 无需安装 numpy/pandas（纯 Python 实现）
- 内网部署无障碍（除 openpyxl 外零依赖）
